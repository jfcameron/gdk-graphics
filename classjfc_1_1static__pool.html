Detailed Description

template<typename value_type, const size_t pool_size>
class jfc::static_pool< value_type, pool_size >

Object pool of a fixed pool_size.

When get is called, an object with a reference count of 1 (unused outside the pool) is returned. If no unused objects exist, a nullptr is returned.
