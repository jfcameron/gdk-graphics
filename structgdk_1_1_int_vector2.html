Detailed Description

template<typename component_type_param = int>
struct gdk::IntVector2< component_type_param >

Behaves like a Vector2 but uses integral types for storage rather than floating point types.

Useful for describing such things as non-normalized texel position, tilegrid position, window size, etc.
