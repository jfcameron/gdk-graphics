Detailed Description

pod struct representing decoded image data + metadata that describes a 2d image primary purpose is used in construction of a texture object. a view does not own its data. The user must ensure the data observed by the view is cleaned up sometime after it has been used. By used, generally I mean sometime after it has been passed to a texture ctor
