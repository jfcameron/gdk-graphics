template<typename component_type_param = float>
struct gdk::Vector2< component_type_param >

2 component floating point struct.

useful for storing 2D position, speed, direction, normalized coordinates.

Note
in reference to the static members: X is considered the lateral component, pointing right, Y the vertical, pointing up
